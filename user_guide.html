ユーザーガイド / User Guide

  1. トップページの配送エリアをご確認くださいませ。
  2. 配送エリア内はご購入代金合計600baht以上で送料無料で配送サービスを承っております。※お買い物合計金額600baht以下の配送は行っておりません。
  3. 会員登録を完了してからご注文をお願いいたします。お支払いは商品到着時にタイバーツ現金にてお支払いいただくか、銀行振込みをご利用いただけます。※口座番号等々はご注文完了時の自動配信メール内にてご案内しております。
  4. 配送エリア外への配送に関しましてはお問い合わせくださいませ。
  5. 県外への配送も可能です。SCGヤマト様のクール便を利用配送料はSCGヤマト様の実費を頂いております。別途発泡スチロール箱やドライアイスの手配も別料金にて可能です。
  6. 配送の混み具合により早期に当日の配送受付を締め切る場合が御座います。
  7. webサイトからご購入の場合はご購入金額の約1%のポイントが取得でき次回のお買い物の際に1ポイント＝1bahtとしてご利用いただけます。最後にご利用いただいてからご利用がないまま90日が経過しますとポイントは失効となりますのでご注意くださいませ。また、ポイントの換金は致しかねます。
  8. タイの国民の祝日及び毎週日曜日、第三土曜日を定休日とさせて頂いております。
